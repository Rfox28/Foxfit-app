# Fox Fit — Training Log

Personal training app. Open source, self-hosted.

## Install on iPhone
1. Open your GitHub Pages URL in Safari
2. Tap Share → Add to Home Screen
3. Tap Add

## Republish after changes
1. Drop updated files into this folder (replacing old ones)
2. Run: `bash publish.sh`
3. Wait ~1 min for GitHub Pages to update

## Files
- `index.html` — the full app
- `sw.js` — service worker (offline support)
- `manifest.json` — app metadata (name, icon, colours)
- `icon-192.png` / `icon-512.png` — home screen icons
- `publish.sh` — one-command deploy script
